# Chevrolet Camaro ZL1 — Interactive Showcase

An immersive, cinematic front-end experience inspired by the Chevrolet Camaro ZL1. This project recreates a premium automotive product website using modern web technologies, combining smooth parallax scrolling, scroll-triggered animations, and layered visual storytelling to deliver an engaging user experience.

## ✨ Highlights

* 🚗 Cinematic full-screen hero section
* 🌄 Multi-layer parallax scrolling effects
* 🎬 Scroll-triggered animations and content reveals
* ⚡ Animated performance statistics
* 🎨 Premium automotive-inspired UI/UX
* 📱 Responsive layout
* 🖼️ High-quality imagery with depth and motion
* 🚀 Smooth section-to-section transitions

## 🛠️ Technologies

* HTML5
* CSS3
* JavaScript
